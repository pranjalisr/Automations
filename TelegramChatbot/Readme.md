# 🤖 Telegram AI Chatbot — n8n Automation

A smart AI-powered Telegram chatbot built with n8n using OpenAI Agents, conversational memory, calculator tools, and external search capabilities.
This workflow enables natural conversations directly inside Telegram while supporting tool calling and contextual responses.

## 🚀 Features

* 💬 Telegram chatbot integration
* 🤖 OpenAI-powered AI Agent
* 🧠 Conversational memory support
* 🧮 Built-in calculator tool
* 🔎 Search agent integration
* ⚡ Real-time AI responses
* 🛠 Tool-calling agent architecture
* 🔄 Context-aware conversations

## 🏗 Workflow Overview

1. A user sends a message through Telegram.
2. The Telegram Trigger activates the workflow.
3. The message is sent to the AI Agent.
4. The AI Agent uses:

   * OpenAI Chat Model for reasoning
   * Simple Memory for context retention
   * Calculator tool for computations
   * Search Agent for external information retrieval
5. The AI generates a contextual response.
6. The reply is automatically sent back to Telegram.
