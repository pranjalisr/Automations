# n8n Automations

A growing collection of AI-powered n8n workflows designed to automate real-world tasks using APIs, AI agents, webhooks, and third-party integrations.

This repository contains reusable automations for productivity, assistants, news workflows, integrations, and experimental AI systems — with many more workflows being added continuously.

## Current Workflows

- **MCP Assistant** – AI assistant workflow with tool-based interactions.
- **Productivity Assistant** – Automates productivity and task management flows.
- **Telegram Assistant** – AI-powered Telegram interaction workflow.
- **Chat with News** – Fetches, summarizes, and enables conversations with latest news.

## Goal

The goal of this repository is to build production-ready and reusable automations that simplify repetitive tasks, improve workflows, and explore practical AI integrations.

## How to Use

1. Clone the repository
2. Open n8n
3. Import any workflow JSON file
4. Configure credentials/API keys
5. Activate and run the workflow
