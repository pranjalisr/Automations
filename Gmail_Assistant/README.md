# 📧 AI Gmail Assistant — n8n Workflow

An AI-powered Gmail assistant built with n8n that enables conversational email automation using OpenAI Agents, memory, and Gmail integration.

This workflow allows users to interact with an AI agent through chat and automatically send emails using natural language commands.

## 🚀 Features

* 💬 Conversational AI interface
* 🤖 OpenAI-powered AI Agent
* 📧 Gmail email sending automation
* 🧠 Context-aware memory support
* ⚡ Natural language email commands
* 🛠 Tool-calling AI architecture
* 🔄 Real-time AI interactions
* 📚 Persistent conversational context

## 🏗 Workflow Overview

1. A user sends a chat message.
2. The AI Agent receives the input.
3. The agent uses:

   * OpenAI Chat Model for reasoning and response generation
   * Simple Memory for conversation context
   * Gmail Send Tool for email automation
4. The AI interprets user intent and sends emails when requested.
5. The workflow returns intelligent conversational responses.
