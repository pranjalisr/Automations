# 🤖 WhatsApp AI Assistant — n8n Automation

An AI-powered WhatsApp assistant built with n8n that enables intelligent conversations, contextual memory, calculations, Wikipedia search, and Gmail automation directly from WhatsApp.

This workflow transforms WhatsApp into a smart AI assistant capable of answering questions, performing tasks, retrieving information, and sending emails through natural conversation.

## 🚀 Features

* 💬 WhatsApp chatbot integration
* 🤖 OpenAI-powered AI Agent
* 🧠 Conversational memory support
* 🧮 Built-in calculator tool
* 📚 Wikipedia search integration
* 📧 Gmail sending capability
* ⚡ Real-time AI responses
* 🔄 Tool-calling agent architecture
* 🛠 Context-aware conversations

## 🏗 Workflow Overview

1. A user sends a WhatsApp message.
2. The WhatsApp Trigger activates the workflow.
3. The message is passed to the AI Agent.
4. The AI Agent uses:

   * OpenAI Chat Model for reasoning
   * Simple Memory for maintaining conversation context
   * Calculator tool for computations
   * Wikipedia for factual information retrieval
   * Gmail tool for sending emails
5. The AI generates an intelligent response or performs requested actions.
6. The final reply is sent back to WhatsApp automatically.
